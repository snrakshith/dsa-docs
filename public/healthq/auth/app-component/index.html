App Component Overview#

This component serves as the root entry point of the application, setting up essential providers and routing to ensure state management, authentication, and navigation work seamlessly.

Code Overview#

How it Works#

  1. StrictMode:

    • Helps identify potential issues in the React code during development.

  2. Suspense:

    • Displays SplashScreen as a fallback while components or routes are being loaded.

  3. AuthProvider:

    • Provides OIDC authentication using the configured userManager and callback for sign-in events.

  4. Router:

    • Wraps the application in React Router to enable navigation between routes.

  5. Provider:

    • Connects the Redux store to the app to manage global state.

  6. CustomRoutes:

    • Defines the routes and components rendered based on the user’s path in the browser.

Summary#

This component ties together state management, authentication, and routing, ensuring a smooth user experience throughout the app.
