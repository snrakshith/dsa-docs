PieChart Component Documentation

The PieChart component is a reusable and interactive pie chart built using the @visx/shape, @visx/group, and @visx/tooltip libraries. This component displays aggregated data visually as pie slices, with legends and tooltips to enhance data presentation and user interaction.

Component Props#

  • aggregatedData:
    • Type: DataPoint[]
    • Description: An array of data points containing the name, value, and color for each pie slice.
    • Example:

Core Features#

  • Dynamic Legend:
    • Automatically displays each category name with a color indicator.
  • Interactive Tooltips:
    • Hovering over a slice shows a tooltip with detailed information.
  • Hover Effects:
    • Slices reduce opacity when hovered to highlight the selected section.
  • Responsive Positioning:
    • The legend is placed flexibly beside the chart to improve layout aesthetics.

Dependencies#

Code Breakdown#

1. Colors and Memoized Data Preparation#

A list of predefined colors is used to assign colors to each pie slice, looping through them using the index. A memoized agData object ensures the data preparation is efficient and only recalculates when aggregatedData changes.

2. Tooltip Handling#

The useTooltip hook manages the visibility and positioning of tooltips. localPoint extracts the mouse coordinates from the event for precise tooltip placement.

3. SVG Pie Chart Structure#

The Pie component creates arcs for each data point. Each arc is styled to change opacity on hover, with the slice’s name displayed at the centroid.

4. Legend Display#

The legend is displayed to the right of the pie chart, with each category represented by a colored box and its corresponding name.

5. Tooltip Rendering#

A tooltip is conditionally rendered near the mouse pointer when hovering over a slice, showing the name and value of the selected category.

Styling and Layout#

  • Chart and Legend Alignment: The chart is centered horizontally, and the legend is aligned beside it using flexbox.
  • Opacity Changes: The slices’ opacity changes on hover to improve interactivity.
  • Tooltip Colors: The tooltip has a black background with white text for contrast.

Usage Example#

Conclusion#

The PieChart component is an intuitive, visually appealing way to represent aggregated data. It supports interactive features such as tooltips and legends, making it highly suitable for dashboards and data visualization applications.
