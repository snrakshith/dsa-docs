title: "21. Sum of Even Numbers in a Range"
description: ""
---

### Solution

```js
const sumEvenNumbersInRange = (start, end) =>
  Array.from({ length: end - start + 1 }, (_, i) => i + start)
    .filter((n) => n % 2 === 0)
    .reduce((a, b) => a + b, 0);
console.log(sumEvenNumbersInRange(1, 10)); // 30
```
